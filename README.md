# 🌐 Laboratorios y Prácticas de Cisco Packet Tracer

Este repositorio aloja una colección de topologías, escenarios de simulación y configuraciones avanzadas desarrolladas en la plataforma **Cisco Packet Tracer**, enfocadas en el diseño de redes, enrutamiento, conmutación y seguridad perimetral.

---

# 📊 Aspectos Destacados de las Prácticas

🔒 **Seguridad de Dispositivos:** Implementación de autenticación, contraseñas encriptadas y el protocolo AAA.
🛠️ **Administración del CLI:** Configuración nativa en Cisco IOS para routers, switches y terminales.
🌐 **Servicios de Red:** Despliegue y simulación interactiva de servidores HTTP, DNS, DHCP y direccionamiento IPv4/IPv6.

---

# ⚙️ Estructura del Repositorio

Cada práctica cuenta con su respectiva documentación y archivos de simulación organizados de la siguiente manera:
*   `📁 Archivos_PKT/`: Archivos fuente de simulación listos para ejecutarse en Packet Tracer.
*   `📁 Scripts_Configuracion/`: Respaldo en texto plano (.txt) de los comandos IOS aplicados en las consolas de routers y switches.
*   `📁 Topologias/`: Capturas visuales y esquemas del diseño físico y lógico de las redes.

---

## 🚀 Cómo utilizar estos laboratorios
1. Asegúrate de tener instalado **Cisco Packet Tracer** (versión 8.0 o superior recomendada).
2. Clona este repositorio o descarga la práctica que desees revisar.
3. Abre el archivo `.pkt` desde la carpeta correspondiente y accede a las terminales CLI para examinar los scripts de configuración.
